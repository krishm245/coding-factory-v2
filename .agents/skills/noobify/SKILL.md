---
name: noobify
description: Explain a certain topic to the user like they are an absolute beginner in that topic. Do NOT assume that they know anything about the topic. Use simple language and avoid jargon. If you need to use jargon, explain it first.
---

Explain the topic to the user in simple language, avoiding jargon. If you need to use jargon, explain it first. Do not assume that the user knows anything about the topic. Break it down into small easy to follow steps, and use relevant examples to illustrate the concept.
